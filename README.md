# Менеджер проектов - Telegram Bot
## Описание
Бот для управления личными проектами. Позволяет сохранять информацию о проектах, отслеживать статусы и добавлять навыки.

## Основные команды
/start - Начать работу с ботом

/info - Показать список команд

/new_project - Создать новый проект

/projects - Показать все проекты

/update_projects - Изменить информацию о проекте

/delete - Удалить проект

/skills - Добавить навыки к проекту

/add_description - Добавить описание проекта

/manage_statuses - Управление статусами проектов

/manage_skills - Управление навыками

## Функционал
Создание и удаление проектов

Добавление описания, ссылок и статусов

Управление навыками проектов

Просмотр информации о проектах

Обновление данных проектов
